## Run local version
```
npm run dev
```
Open `http://localhost:8080/`
### Start dev client:
```
npm run dev:client
```
Client host `http://localhost:8080/`
### Start dev server:
```
npm run dev:server
```
Server host `http://localhost:8088/`
## Run prod version
```
npm run prod
```
Open `http://localhost:8080/`
